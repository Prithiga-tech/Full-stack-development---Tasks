# TechNova Landing Page

## Project Overview

TechNova is a modern landing page website created using HTML and CSS. This is my first website project as part of my Full Stack Development learning journey. The website showcases a technology company with sections such as Home, Features, About Us, and Contact.

## Features

* Responsive Navigation Bar
* Hero Section with Background Image
* Features Section with Multiple Cards
* About Us Section
* Contact Form
* Smooth Scrolling Navigation
* Flexbox Layout
* Parent-Child CSS Selectors
* Clean and Modern UI Design

## Technologies Used

* HTML5
* CSS3
* Flexbox
